How to install

Note: The installation/configuration process will be more user-friendly in the future. 😅

The following steps need to be completed no matter which toy integration you intend to use afterwards.

  1. Download and install Python (version 3.7 or later?) on your machine. Python is available for free here: https://www.python.org/downloads/windows/.

  2. Download Skyrim Toy Interface from GitHub [direct download link] or LoversLab and extract the ZIP file to a folder. The unzipped contents should look like this: 

    .
├── common
│...
├── pattern_generator
│...
├── plugin
│   ├── scripts
│   │   ├── SkyrimToyInterfacePlayerScript.pex
│   │   └── source
│   │       └── SkyrimToyInterfacePlayerScript.psc
│   └── SkyrimToyInterface.esp
├── settings.py
├── SkyrimToyInterface.py
└── toys
    ├── ...

  3. Install the Skyrim mod SkyrimToyInterface.esp which is found in the plugin folder. The easiest way to install the mod is to zip the contents of the folder (both the ESP file and the scripts folder!) into a new ZIP archive and then load & install it through a mod manager like Nexus Mod Manager or Vortex.

    TIP: You can verify that the mod is running by typing 

    help SkyrimToyInterfaceControllerQuest
    in the in-game console. The mod should show up as a loaded quest like this: 
    QUST: SkyrimToyInterfaceController

  4. Open settings.py in an ordinary text editor (like notepad++ or VS Code) and change the variables LOG_PATH and CHARACTER_NAME in the file:

    from common.constants import *
## Edit the following variables to configure this script

# Important variables to configure for all users:
LOG_PATH = '../Documents/My Games/Skyrim VR/Logs/Script/Papyrus.0.log' # Full path to the Papyrus log that the script should parse
CHARACTER_NAME = "Min"  # The name of your character.

    LOG_PATH Should point to the place on your hard disk where Skyrim outputs its Papyrus log file. The precise file path depends on your particular Skyrim installation, but for most installations the log can usually be found in 

    C:/Users/YOUR_WINDOWS_ACCOUNT_NAME/Documents/My Games/Skyrim/Logs/Script/Papyrus.0.log

    CHARACTER_NAME must correspond to the name of your in-game character. This ensures that the script can parse the correct information from the log file.

    Caution: settings.py is a PY (python script) file. Make sure your text editor does not save it as a TXT file like settings.py.txt!

  5. Enable debug tracing in your papyrus.ini[?] configuration file. Papyrus should have these two flags enabled. This allows Papyrus to output debug information into the log, which the script needs to function:

    [Papyrus]
bEnableLogging=1
bEnableTrace=1
